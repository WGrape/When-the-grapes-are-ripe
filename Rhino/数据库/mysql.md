### 四大隔离级别
1. Read uncommitted 读未提交：读取未提交内容，什么都没解决 。
2. Read committed   读提交：读取提交内容，解决脏读 。
3. Repeatable read  可重复读：解决不可重复读 。 （ MySQL默认隔离级别 ）
4. Serializable     可串行化：解决所有问题 。
