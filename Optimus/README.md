## by Zephir
Zephir是 Phalcon 团队下的一个开源产品，旨在为PHP开发者提供一种简单快捷的方式开发PHP扩展 。
Zephir是一种动态静态结合的运行在Zend引擎下的编译类型的语言，语法和PHP有很大的相似之处，适合PHP程序员使用。
