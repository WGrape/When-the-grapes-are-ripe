# NarniaOS
操作系统的简单实现（A simple implementation of an OS）.

## 介绍（Introduction）
Linux作为优秀的开源操作系统，其最早期版本(0.11)非常值得学习，故此项目是以Linux的0.11版本为最重要的基础，在学习和研究此版本的代码基础上加以自己的想法和设计而实现的。
